# light-join

Join urls or paths into single path.

[![NPM][npm-image]][npm-link]
[![Minzipped Bundle Size][bundlephobia-image]][bundlephobia-link]

## Install

with npm
```
npm i -s light-join
```

with yarn
```
yarn add light-join
```

## Usage

JavaScript
```javascript
import lightJoin from 'light-join';

lightJoin('/your-base/url/', 'some/path', '/some-other/path');
```
