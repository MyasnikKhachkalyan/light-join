# light-join

Join urls or paths into single path.

## installation

with npm
```
npm i -s light-join
```

with yarn
```
yarn add light-join
```

## usage

JavaScript
```javascript
import lightJoin from 'light-join';

lightJoin('/your-base/url/', 'some/path', '/some-other/path');
```

TypeScript
```typescript
import lightJoin from 'light-join';

lightJoin('/your-base/url/', 'some/path', '/some-other/path');
```
